## Config Condition
Checks if the provided config option matches the provided value

### Properties
| Name | Type | Default |
|--------|-------|---------|
| Config | String | N/A |
| Value | Unknown | defaults to `true` if bool, defaults to empty if string |

### Usage
```json
{
    "type": "config",
    "config": "Replacements:replace_spider_sounds"
}
```
