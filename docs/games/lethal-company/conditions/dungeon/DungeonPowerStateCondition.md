## Dungeon Power State Condition
Checks if the current dungeon has power. This includes players flipping the breaking, or pulling the apparatus.

### Properties
| Name | Type | Default |
|--------|-------|---------|
| Value | Boolean | `true` |

### Usage
```json
{
    "type": "LethalCompany:dungeon:power_state",
    "value": "false"
}
```
