## Player Standing On Condition
Checks what type of ground the current player is standing on. These values can be found in a sound report under "Found Footstep Surfaces".

### Properties
| Name | Type | Default |
|--------|-------|---------|
| Value | String | N/A |

### Usage
```json
{
    "type": "LethalCompany:player:standing_on",
    "value": "aluminum"
}
```
