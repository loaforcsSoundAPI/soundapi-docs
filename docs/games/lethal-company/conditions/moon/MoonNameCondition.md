## Moon Name Condition
Check the current name of the moon. If you want to check multiple use the [or](/soundpack-api/conditions/OrCondition) condition.

### Properties
| Name | Type | Default |
|--------|-------|---------|
| Value | String | N/A |

### Usage
```json
{
    "type": "LethalCompany:moon:name",
    "value": "AssuranceLevel"
}
```
